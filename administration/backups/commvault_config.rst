Commvault Settings
------------------

Enabled
  Enable the Commvault integration
Default Backup Provider
  Sets Commvault as the Default Backup Provider in |conduit| . Backup Providers can be configured per Instance backup settings.
Host
  IP or Hostname of the Commvault server.
Port
  Port configured to access the Commvault server
Commvault server Username
  Admin Username for Commvault
Password
  Password for Username provided (encrypted in |conduit| )
