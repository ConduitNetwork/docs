
Installation
============

|conduit| comes packaged as a ``debian`` or ``yum`` based package. It can be installed on a single on/off premise linux based host or configured for high availability and horizontal scaling. |conduit| is currently only supported on Ubuntu 14.04, Ubuntu 16.04 , CentOS 7.0 or newer, and RHEL 7.0 or newer based hosts.
